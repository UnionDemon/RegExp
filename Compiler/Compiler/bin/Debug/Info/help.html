Меню текстового редактора

Пункт "Файл" меню текстового редактора

В пункте "Файл" меню текстового редактора реализован следующий функционал (см. рисунок 1):

  • Создание документа
  • Открытие документа
  • Сохранение текущих изменений в документе
  • Сохранение документа в новый файл
  • Выход из текстового редактора


Пункт "Правка" меню текстового редактора

В пункте "Правка" меню текстового редактора реализован следующий функционал (см. рисунок 2):

  • Отмена изменений
  • Повтор последнего изменения
  • Вырезать текстовый фрагмент
  • Копировать текстовый фрагмент
  • Вставить текстовый фрагмент
  • Удалить текстовый фрагмент
  • Выделить все содержимое документа


Пункт "Задание" меню текстового редактора

При вызове команд этого меню происходит запуск алгоритмов поиска подстрок, а также справка по лабораборной работе "Изучение регулярных выражений".

Пункт "Пуск" меню текстового редактора

При нажатии на пункт "Пуск" происходит запуск синтаксического анализатора текста - алгоритма поиска подстроки для задания 12 (см. рисунок 3).



Пункт "Справка" меню текстового редактора

Приложение имеет справочную систему, запускаемую командой «Вызов справки» (см. рисунок 4).

Справка содержит описание всех реализованных функций меню.(см. рисунок 5)





Пункт "Локализация" меню текстового редактора

Интерфейс позволяет пользователю выбирать язык, на котором он хочет работать (см. рисунок 6, а, б).

Панель инструментов текстового редактора

Панель инструментов содержит кнопки вызова часто используемых пунктов меню:

  • Создание документа (см. рисунок 8, а)
  • Открытие документа (см. рисунок 8, б)
  • Сохранение текущих изменений в документе (см. рисунок 8, в)
  • Отмена изменений (см. рисунок 8,г)
  • Повтор последнего изменения (см. рисунок 8, д)
  • Копировать текстовый фрагмент (см. рисунок 8, е)
  • Вырезать текстовый фрагмент (см. рисунок 8, ж)
  • Вставить текстовый фрагмент (см. рисунок 8, з)
  • Запуск синтаксического анализатора - алгоритма поиска подстроки для задания 12 (см. рисунок 8, и)

Дополнительные возможности текстового редактора

  1. Интерфейс позволяет работать с несколькими текстовыми документами одновременно (см. рисунок 9, а, б).


  2. Интерфейс позволяет открывать файл при перетаскиваниии его иконки в область редактирования.
  3. Нумерация строк в окне редактирования текста (см. рисунок 10).
