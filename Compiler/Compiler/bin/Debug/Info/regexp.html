Задача 19.

Условие задачи:

На основании свойств регулярных выражений доказать тождества для произвольных R1, R2:

R2 | R2R1 | R2R1* = R2R1*

Решение:

R2 | R2R1 | R2R1* = R2R1*

По свойству 4 "Операция конкатенации дистрибутивна":

R2 (ε | R1 | R1*) = R2R1*

По определению R1* = ε | R1 | R12 | ... | R1n, следовательно:

R2R1* = R2R1*, что и требовалось доказать.

Для поиска в тексте подстрок пусть R1 = 1*2, R2 = ab*.

Опишем язык, порождаемый регулярным выражением: R2R1*:

L(G19)={(abl)(1n2)k, l ≥ 0, n ≥ 0, k ≥ 0}.
